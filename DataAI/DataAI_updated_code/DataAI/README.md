# BNS Legal Assistant

AI chatbot for Bharatiya Nyaya Sanhita 2023 using RAG + FAISS.

## How to Run
1. Open Databricks → Import notebook
2. Run all cells
3. Interactive chat UI appears
4. Ask questions like "What is punishment for rape?"

## Features
- FAISS vector search
- 64 BNS sections
- Chat interface
